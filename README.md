# 自種自售

這是一個專門給自家農場品上架的店商網頁，目前僅供展示使用，沒有實際的交易功能，這個網頁是集結了我目前(2025/2)會所有技術，網頁架構是前後端分離，這個是前端的部分。

## 專案安裝

```bash
pnpm install

```

## 專案啟動

```bash
pnpm dve
```
